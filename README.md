**ATTENTION:** **This bot uses the new version of  [discord.py v1.0+](https://github.com/Rapptz/discord.py).**
This is a german chat bot designed for a special Server and usecase.
If you are looking for a python discord bot to host for yourself and and customizable for your purposes, you should rather take a look at [Red Bot](https://github.com/Twentysix26/Red-DiscordBot). TDE is only meant to be run on my own server.

## Installation
<pre><code> # <span class="pl-s1">clone this repo</span>
 $ <span class="pl-s1">git clone https://github.com/D3monEmper0r/CA-Discord-Bot.git<span>

 # <span class="pl-s1">change the working directory to CA-Discord-Bot<span>
 $ <span class="pl-s1"><span class="pl-c1">cd</span> CA-Discord-Bot</span>

 # <span class="pl-s1">install the requirements</span>
 $ <span class="pl-s1">python3 -m pip install -r requirements.txt</span></code></pre>
 **NOTE:** Use the **configs.yaml** file in the **Configs Folder** to add your Bot Token and adjust the required variables.
